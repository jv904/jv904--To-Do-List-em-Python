# To Do List Tkinter

Aplicativo de lista de tarefas desenvolvido em **Python** com interface gráfica usando **Tkinter** e armazenamento local com **SQLite**.

## 🚀 Funcionalidades

* Adicionar tarefas
* Remover tarefas selecionadas
* Armazenamento em banco de dados SQLite
* Interface simples e intuitiva
* Atualização automática da lista

## 🛠️ Tecnologias Utilizadas

* Python 3
* Tkinter
* SQLite3

## 📷 Preview

Interface desktop para gerenciamento de tarefas.

```bash
📁 projeto
 ┣ 📄 Tarefas.py
 ┣ 📄 tarefas.db
 ┗ 📄 README.md
```

## ▶️ Como Executar

1. Clone este repositório:

```bash
git clone https://github.com/jv904/jv904--To-Do-List-em-Python.git
```

2. Acesse a pasta:

```bash
cd jv904--To-Do-List-em-Python
```

3. Execute o projeto:

```bash
python Tarefas.py
```
